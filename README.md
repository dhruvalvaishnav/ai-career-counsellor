# AI Career Counsellor

## Description
AI-Powered Career Counsellor platform to help students discover tailored career guidance based on their aptitude tests.

### Features
- **Backend:** Java Spring Boot microservices.
- **Frontend:** ReactJS for interactive UI.
- **Messaging:** Apache Kafka for streaming test results.
- **Containerization:** Docker for seamless development and deployment.
